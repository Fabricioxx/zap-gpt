# zap-gpt
Bot para WhatsApp Usando ChatGPT

## Part 1

####install
````
npm init
npm install venom-bot
npm install chatgpt
npm install dotenv
````

#### package.json

````javascript
"main": "index.mjs", <- mudar o arquivo index.js para index.mjs

````

#### start
````
node .
````
